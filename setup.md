# Course Project – Performance Audit Report

## Website

**Website Name:** AP News  
**URL:** https://apnews.com/

---

## Why Is This Website a good candidate

 News websites need to load quickly because users expect instant access to breaking news. They also contain a large amount of images, advertisements, videos, and JavaScript, making them a good candidate for a performance audit. By analyzing this website, it is possible to identify how different types of content affect loading speed and user experience.

## Pages Included in the Audit

### 1. Homepage

https://apnews.com/

The homepage is the main entry point for most visitors. It contains many news articles, images, advertisements, and dynamic content, making it useful for measuring the overall performance of the website.

---

### 2. U.S. News

https://apnews.com/us-news

This section contains a large number of articles and images that update frequently. It is useful for evaluating how category pages perform.

### 3. World News

## Repository Purpose

This repository will contain the performance audit for AP News using tools such as Google PageSpeed Insights and Chrome DevTools. The report will analyze loading performance, network activity, Core Web Vitals, caching, resource optimization, and provide recommendations for improving the site's speed and user experience.
